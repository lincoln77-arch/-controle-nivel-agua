# controle-nivel-agua
# 🌊 Controle de Nível de Água

Projeto simples desenvolvido em Python com o objetivo de simular o monitoramento do nível de água de um reservatório.

O sistema apresenta:
- níveis de água em diferentes estados;
- indicador visual em barra;
- alertas de nível crítico e reservatório cheio;
- exibição colorida utilizando a biblioteca Colorama.

---

## 💡 Objetivo do Projeto

A proposta deste projeto é demonstrar conceitos básicos de programação, como:

- estruturas condicionais;
- funções;
- listas;
- repetição;
- organização de código;
- utilização de bibliotecas externas.

---
